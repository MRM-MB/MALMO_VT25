# Assignment 5

![Assignment 5 Preview](../images/malmoA5.webp)

A Windows Forms application for managing customer contacts.

## Features

*   **Contact Management**: Add, update, and delete contact information.
*   **Address Book**: Store details like address, email, and phone numbers.
*   **Customer Data**: Manage customer-specific data.

## How to Run

1.  Open `Assignment5.sln` or `Assignment5.csproj` in Visual Studio.
2.  Build and Run (`F5`).
