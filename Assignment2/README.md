# Assignment 2

![Assignment 2 Preview](../images/malmoA2.webp)

This assignment covers various utility programs and logic implementations in C#.

## Contents

*   **Assignment 2A (Cost Calculator)**: Calculates costs based on input parameters.
*   **Assignment 2B (Guess The Number)**: A classic number guessing game.
*   **Assignment 2C (String Functions & Temp Converter)**: Utilities for string manipulation and temperature conversion.
*   **Assignment 2D (Scheduler)**: A simple scheduling application.

## How to Run

Navigate to the specific part folder (e.g., `Assignment2A`) and run:

```bash
dotnet run
```
