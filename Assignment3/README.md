# Assignment 3

![Assignment 3 Preview](../images/malmoA3.webp)

A Windows Forms application designed to perform various personal calculations.

## Features

*   **Retirement Savings Calculator**: Estimates savings growth over time.
*   **Water Intake Calculator**: Calculates recommended daily water intake.
*   **BMI Calculator**: Calculates Body Mass Index based on personal data.

## How to Run

1.  Navigate to the project folder:
    ```bash
    cd Assignment3
    ```
2.  Run the application:
    ```bash
    dotnet run
    ```
