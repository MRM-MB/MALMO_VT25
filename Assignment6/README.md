# Assignment 6

![Assignment 6 Preview](../images/malmoA6.webp)

A Windows Forms application for managing tasks with file persistence.

## Features

*   **Task Management**: Create, prioritize, and track tasks.
*   **File Handling**: Save and load tasks from files.
*   **Priority System**: Assign priorities to tasks.
